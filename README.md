## Getting Started

To run this portfolio locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/roseandlily33/reactPortfolio-olio.git
   cd reactPortfolio-olio
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

All data and logic are handled on the frontend; there is no backend or database.
